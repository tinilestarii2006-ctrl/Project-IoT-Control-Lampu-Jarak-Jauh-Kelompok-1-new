# PROJECT IoT: Kontrol Lampu Jarak Jauh

## Deskripsi
Project ini digunakan untuk mengontrol lampu dari jarak jauh menggunakan NodeMCU dan WiFi.

## Software Preparation
1. Arduino IDE
2. Blynk

## Hardware Preparation
1. ESP32
2. Relay 
3. Lampu dan Steker
4. Breadboard
5. Kabel Jumper Fiemale to Fiemale
6. Colokan

## Library & Anything
1. Blynk (By Volodymyr Shymanskyy)
2. WiFi (Built-in ESP32)
3. HTTPClient
## Cara Kerja
1. NodeMCU terhubung ke WiFi
2. HP membuka web server
3. Klik ON/OFF
4. Lampu menyala/mati

## Hasil
Lampu dapat dikontrol melalui HP dari jarak jauh.
